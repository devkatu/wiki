クラス、ID等のcss的な指定

クラス、ID等はcssと同様に指定できる。 クラスは . IDは # を先頭に付ける。 しかしcssで言う疑似クラス(:nth-childとか)はjqueryではフィルタとなっており、表現の仕方が若干違う。 フィルタのセレクタとフィルタ の:(コロン)の間をあけないように！！開けるとスペースと なるのでその下の要素とみなされるよ!!

' '(スペース) : その下層の要素すべて

'>' : その直下の要素のみ

':eq(n)' : その層のn番目の要素

':gt(n)' : その層のn番より後の要素(自分を含まない)

':lt(n)' : その層のn番より前の要素(自分を含まない)

$('.box_chg').css('background-color','green');

$('#box1-1').css('background-color','blue');

$('#section1 > div > div:eq(2)').css('background-color','purple');

$('#section1 > div > div:gt(2)').css('background-color','orange');

$('#section1 > div > div:lt(2) p').css('background-color','yellow');

メソッドによるDOM要素の指定(トラバース)

'parent()' : セレクタの親要素

'children()' : セレクタの子要素

'next()' : セレクタの次の要素

'prev()' : セレクタの前の要素

'siblings()' : セレクタの兄弟の要素

$('#bsli').parent().css('border','solid 5px pink');

$('#bsli').children().css('border','dotted 2px blue');

$('#bsli').next().css('border','double 3px red');

$('#bsli').prev().css('border','dashed 2px black');

$('#bsli').siblings().css('background','yellow');

要素の挿入、挿入された要素に対するイベントハンドラ設定、属性の設定


        $('#before').click(function(){
          var elm = $('<ul><li>1-1-1</li></ul>').css('color','red');
          $('#elm1_2').before(elm);
        });

        $('#after').click(function(){
          var elm = $('<ul><li>1-2-1</li></ul>').css('color','red');
          $('#elm1_2').after(elm);
        });

        $('#ibefore').click(function(){
          var elm = $('<ul><li>2-1-1</li></ul>').css('color','red');
          elm.insertBefore('#elm2_2');
        });

        $('#iafter').click(function(){
          var elm = $('<ul><li>2-2-1</li></ul>').css('color','red');
          elm.insertAfter('#elm2_2');
        });

        $('#prepend').click(function(){
          var elm = $('<li>').text('3-0').css('color','red');
          $('#elm3').prepend(elm);
        });

        $('#append').click(function(){
          var elm = $('<li>').text('3-4').css('color','red');
          $('#elm3').append(elm);
        });

        $('#prependto').click(function(){
          var elm = $('<li>').text('4-0').css('color','red');
          elm.prependTo('#elm4');
        });

        $('#appendto').click(function(){
          var elm = $('<li>').text('4-4').css('color','red').attr('id','elm4last');
          elm.appendTo('#elm4');
        });

        $('body').on('click','#elm4last',function(){
          alert('this element was added by "on" method!!');
        });

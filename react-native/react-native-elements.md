## icon コンポーネント
icon使うなら
[ここ](https://oblador.github.io/react-native-vector-icons/)
を参照して

```
    <Icon
        type="【アイコンのタイプ(赤いラベルの所)】"
        name="【探したアイコンの名前】"
        ...
    />

```
のようにするといい

ロールオーバー(画像の差替)


$('a>img').each(function(i,e){
  var off = $(e).attr('src');
  var on = off.replace(/^(.+)-off(\.[^\.]+)$/, '$1-on$2');
  $(e).hover(function(){
    $(this).attr('src',on);
  },function(){
    $(this).attr('src',off);
  });
});

cssのhoverではimgタグの切替は不可。 (background-imageとかはできるけど)

上、他方向へスクロールして戻るボタン


$('#back').hide();
$(window).scroll(function(){
  $('#pos').text($(this).scrollTop());
  if($(this).scrollTop() > 60){
    // $('#back').show();
    $('#back').fadeIn();
  }else{
    $('#back').fadeOut();
  }
});
$('#back a').click(goTop);
$('.scrollTop').click(goTop);
function goTop(){
  $('body,html').animate({  // bodyだけ、htmlだけでも効くけどブラウザによるので注意
    scrollTop:0
  },500);
  return false;
}

$('a[href^="#"]').click(function(){
  var href = $(this).attr('href');
  var pos = $(href).offset().top;
  $('body,html').animate({scrollTop:pos},1000);
  return false;
});

初期状態では#back(上へ戻るボタン…画面右下に 固定で出るようにしている)は隠しておき、ある程度スクロール したら出現させるようにwindwo.scrollイベントで 定義しておく。


これ→上へ戻るでも上に戻れるよ！

// ハンバーガーメニュー
$('.Toggle').click(function(){
  $(this).toggleClass('active');
  // .Toggleにactiveクラスが付いているかで分岐
  if($(this).hasClass('active')){
    $('.NavMenu').addClass('active');
  }else{
    $('.NavMenu').removeClass('active');
  }
});

/* ハンバーガーメニュー用css */
/*ナビメニューのスタイルを指定*/
nav.NavMenu{
  position: fixed; /*表示位置を固定*/
  z-index: 2; /*重ね順を変更*/
  top: 0; /*表示位置を指定*/
  left: 0; /*表示位置を指定*/
  background: #fff;/*背景を白にする*/
  color: #000; /*文字色を黒にする*/
  text-align: center; /*テキストを中央揃え*/
  width: 100%; /*全幅表示*/
  transform: translateY(-100%); /*ナビを上に隠す*/
  transition: all 0.6s; /*アニメーションの時間を指定*/
}
nav.NavMenu ul{
  background: #ccc; /*背景をグレーにする*/
  width: 100%;
  margin: 0 auto;
  padding: 0;
}
nav.NavMenu ul li{
  font-size: 1.1em;
  list-style-type: none;
  padding: 0;
  width: 100%;
  border-bottom: 1px dotted #333;
}
nav.NavMenu ul li:last-child{
  padding-bottom: 0;
  border-bottom: none; /*最後のメニュー項目のみ下線を消す*/
}
nav.NavMenu ul li a{
  display: block; /*クリックできる領域を広げる*/
  color: #000;
  padding: 1em 0;
}
/*トグルボタンが押されたときに付与するクラス*/
nav.NavMenu.active{
  transform: translateY(0%);

position:relative;

staticを指定したときの位置(つまりその要素の初期値)を基準にtop,left等で場所を指定する。通常のレイアウト処理から除外される(存在していないかのように扱われる)ため、親要素の高さ、幅は変わらないみたい…親要素にheight、widthを指定する等必要あり

通常の要素後続要素(relative)

position:absolute;

親要素に対する絶対値でtop,left等を指定する。ただし親要素にposition:static;以外が指定されていなければならず 指定無しの場合は更に先祖の要素まで(body要素まで)遡る。通常のレイアウト処理から除外される(存在していないかのように扱われる)ため、親要素の高さ、幅は変わらないみたい…親要素にheight、widthを指定する等必要あり

height,widthの指定方法

基本的にはデフォルトでblock要素、inline-block要素においてwidthは親要素の幅いっぱい、heightは子要素の高さに合わせ成り行き。
inline要素においてwidthは子要素の幅に合わせ成り行き、heightも同様。
※pxはそのままなので例を省略！

width ％指定

親要素の幅(padding含まないcontentの幅)に対する割合い％で指定する。(親要素がbox-sizing:border-box;でもcontentの幅に対する割合となる。)合計が100%を超えないようにする。ただしデフォルトではbox-sizing:content-box;なのでコンテンツ幅が親要素いっぱいになる+paddingなので,親要素からはみ出てしまう。auto(初期値)だとpadding含みで親要素の幅いっぱいに調整される。これを解消するためにbox-sizing:border-box;を指定する(autoと同じ挙動になるはずだがレスポンシブ時は width:100%; と box-sizing:border-box; の方がいいかも)か、paddingとwidthを含め100%となるように指定するとよい。min-width,max-widthを指定しておくとレスポンシブ

50%

30%

20%

auto

100% + content_box(初期値)

100% + border_box

width80% + padding 10%ずつ + content_box

height ％指定

親要素の高さ(padding含まず)に対する割合い％で指定する。合計が100%を超えないようにする。heightは初期値がautoなので、親要素には必ずheight:auto;以外の指定をしなければ子要素の％指定は不可。ただしwidthと同様デフォルトでpaddingを含む高さ指定なので,100%指定するとwidthが親要素の高さいっぱい+paddingで親要素からはみ出てしまう。これを解消するためにbox-sizing:border-box;を指定する。ただしwidthと同様にpaddingとheightを含め100%になるようにはできない(次で説明)。min-width,max-widthを指定しておくとレスポンシブ

50% + border_box

30% + border_box

20% + border_box

100% + content-box(初期値)

100% + border-box

height80% + padding10%ずつ + content-box(レイアウト崩れる)

margin,paddingの％指定は基本横方向のみ

基本横方向のみ％指定で使い、縦方向に使用すると前述のとおりレイアウトが崩れて見える。これは縦方向に％指定しているが、基準となるのは"親要素の高さ"ではなく"親要素の幅"を基準にしているためである。

padding: 25% 0;
横幅の25%ずつを上下にpadding
縦横比を保つことができる。

↑元画像(50px*50px)。以下positionは全てcenter center; top left;とかの指定もでき、親要素に対する位置を指定できる

background-size:auto;

初期値。元画像のサイズそのまま表示する

background-size:contain;

親要素に対し画像を全部表示できる範囲で拡大表示する。

background-size:contain; background-repeat: no-repeat;

繰り返し表示をしないようにする。repeat-x;repeat-y;で横方向、縦方向への繰り返し表示もできる

background-size:cover;

親要素を全て埋め尽くすように拡大表示する。はみ出た部分は非表示

background-attachment:fixed;
